# Python API: CLI-Parity API

Use these modules when you want the same operations as the CLI, but from tests, scripts, notebooks, or another Python process.

This surface is the right fit when you want to:

- load and diff a plan in a test
- render build artifacts from Python
- resolve a resource id to its source or deployed location

The current CLI-aligned helpers center on `plan`, `build`, `deploy`, `map`, `locate`, and `trace` style workflows.
